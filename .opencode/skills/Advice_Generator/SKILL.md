# Advice Generator

Create personalised recommendations using patient data and medical evidence.

## When to use this

Use when the user asks:

* "What should I do?"
* "How can I improve?"
* "What changes would help?"
* "Give me advice."

This skill should use outputs from:

* Data Collector
* Trend Analyzer
* Medical Research

## Procedure

1. Review available patient data.
2. Review trend analysis findings.
3. Review medical evidence.
4. Generate practical recommendations.
5. Explain the reasoning behind each recommendation.
6. Prioritise recommendations by likely impact.
7. Mention uncertainty where appropriate.
8. Include a medical safety disclaimer.

## Recommendation Categories

Possible recommendations include:

* physical activity
* sleep improvements
* nutrition adjustments
* hydration
* stress management
* glucose monitoring habits

## Important

Do not:

* diagnose conditions
* prescribe medications
* recommend changing medication dosages
* replace healthcare professionals

Always remind users that recommendations are educational and not medical advice.
